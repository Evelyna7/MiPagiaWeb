Sobre este sitio

Este proyecto es una demostración de cómo utilizar GitHub Pages para desplegar sitios web estáticos de manera rápida y sencilla.

¿Cómo funciona?

  • Subes los archivos HTML al repositorio de GitHub.
  • Activas GitHub Pages en la configuración del repositorio.
  • Tu sitio queda disponible en una URL pública.
